# Plan: @hermes-forge CLI + TUI + Branding Package

## Goal
Transform `hermes-forge-cli` from a functional-but-basic CLI into a professional, branded terminal experience with a proper TUI dashboard mode, published on npm.

## Current State
- Repo: `github.com/JonasAbde/hermes-forge-cli` (main branch, v1.0.0)
- 20+ commands registered via `commander`
- Output: basic chalk colors, blue headers, simple tables
- TUI: `prompts`-based interactive menus via `forge interactive`
- Deploy: Never published to npm; `publishConfig` points to GitHub Packages
- CI: GitHub Actions workflow exists (self-hosted runner)

## Scope (3 deliverables)

### 1. Branding Package (`src/brand/`)
A self-contained visual identity module used by both CLI and TUI.

**Files:**
- `src/brand/logo.ts` ‚Äî ASCII art logo renderer (multiple sizes)
- `src/brand/colors.ts` ‚Äî Brand color palette (primary indigo, gradients)
- `src/brand/theme.ts` ‚Äî Box drawing, panels, separators, dividers
- `src/brand/spinner.ts` ‚Äî Custom spinner frames
- `src/brand/index.ts` ‚Äî Barrel export

**Design:**
- Primary: `#6366f1` (indigo-500)
- Dark: `#4338ca` (indigo-700)
- Accent: `#8b5cf6` (violet-500)
- Success: `#10b981` (emerald)
- Warning: `#f59e0b` (amber)
- Error: `#ef4444` (red)

### 2. CLI Polish (`src/output.ts`)
Replace the basic output module with branded version.

**Changes:**
- `printHeader()` ‚Üí brand panel with ASCII logo
- `box()` ‚Üí styled panels with gradient borders
- New: `printSection()`, `printKeyValue()`, `printBullet()`
- Spinner integration from brand module
- Consistent help formatting

### 3. TUI Dashboard (`src/tui/`)
Replace `forge interactive` (prompts menu) with a full terminal UI.

**Tech:** `@blessed/neo-blessed` + `blessed-contrib` (mature, no React dep)

**Views:**
- `src/tui/` ‚Äî TUI engine
- `src/tui/dashboard.ts` ‚Äî Main dashboard screen
  - Live service status (API, Web, DB)
  - Pack count + trending
  - Health metrics
- `src/tui/pack-browser.ts` ‚Äî Browse/search packs
- `src/tui/health-monitor.ts` ‚Äî Real-time health dashboard
- `src/tui/scheduler.ts` ‚Äî View/manage scheduled tasks

**Commands:**
- `forge tui` ‚Üí enters TUI dashboard
- `forge tui --mode packs` ‚Üí opens pack browser directly
- `forge tui --mode health` ‚Üí opens health monitor

## Risks & Tradeoffs
- **@blessed/neo-blessed** is a fork of blessed (abandoned). Active but smaller community. Alternative: `ink` (React-based, heavier).
- **TUI = new dependency surface** ‚Äî blessed uses `curses`-like terminal control. `--no-sandbox` may be needed on some environments.
- **npm publish** requires token ‚Äî can be done via `npm publish --access public` or GitHub Packages (existing config).

## Assumptions
- Brand colors follow the Forge web app design tokens (indigo/violet)
- TUI targets Linux/Mac only (no Windows curses)
- npm publish to npmjs.org (not GitHub Packages) for broader access
- User provides npm token for publish step
